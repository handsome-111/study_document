linux：
	指令：
		语法:
			#指令主体 [选项] [操作对象]


		基础指令：
			1.ls: 英文：list，含义：集合，列出当前目录下的所有文件/文件夹名称
				用法：
					1.	ls
					2.	ls 路径		：列出路径下的文件/文件夹

			2.删除：
				rm:
					-i 删除前逐一询问确认。
					-f force,表示强制，不提示是否删除
					-r 删除一个目录，文件夹

				删除文件：rm 文件
				删除文件夹:rm -rf 文件夹

				

			3.编辑文件/打开文件/查看文件:
				vim 文件
					进入文件之后，输入i，开始i编辑，按esc即可退出编辑模式，
						:q!  退出不保存
						:wq  退出兵保存

				cat :直接打开一个文件，查看一个文件，会打印输出在命令行里
					例：
						cat aa.txt		直接查看aa.txt文件

					合并多个文件,查看多个文件：
						cat aa.txt bb.txt 

				head:查看一个文件的前n行，n默认是10
					语法:head -n 文件路径   n表示数字

				tail:查看一个文件的后n行，n默认是10
					语法:tail -n 文件路径   n表示路径

				less：查看文件，以较少的内容进行输出，按下辅助功能键（数字 + 回车 + 空格键 + 上下方向键）查看更多
					语法: less 文件路径
					辅助键：
						回车:一页一页的输出
						空格键:一页一页的输出
						上下键：一行一行的翻

			4.统计文件信息：
				wc:统计文件内容信息（行数，单词数，字节数）
				语法：wc -lwc 文件路径
					-l:Lines,行数
					-w:words,单词数
					-c:字节数


			5.创建文件夹：mkdir 文件夹路径

			6.创建文件:touch 文件路径

			7.移动，剪切：mv

			8.输出重定向:
				本来是在命令行里显示的，但是你可以输出到文件里等，在打开显示都可以，可以当作后台日志等

				>  : 覆盖输出，会覆盖掉原先文件的内容
				>> : 追加输出，不会覆盖原始文件内容，会追加在原始内容的末尾

				例子：
					ls -a > ./aa.txt
					ls -a >> ./aa.txt





		进阶指令：
			1.查看磁盘空间：
				语法:df -h	  -h表示以可读性高的形式展示

			2.查看内存使用情况：
				语法:free -m   -m表示单位为M  以此类推，-g 单位为G   -k 表示kb
				结果：
					total:
					used:
					free:
					shared:
					buffer:系统已经分配且已使用的内存
					cache:系统分配但未被使用的cache 数量
					available:

			3.操作时间,读取/设置时间：
				date:
					语法1:date 
					语法2:date +%F 			输出年月日，2019-01-11
					语法3:date "+%F %T"		输出年月日时分秒
					语法2:date "+%y%m%d"		输出20190111
					语法3:date "+%y-%m-%d"	输出2019-01-11
